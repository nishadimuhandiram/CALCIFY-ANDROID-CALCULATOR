
# CALCIFY. - UWU/PGU/MIT/23/082

Calcify is a simple yet powerful calculator app developed using Java. It leverages the Rhino ScriptEngine to evaluate mathematical expressions, offering users a smooth and efficient calculation experience. Whether you're performing basic arithmetic or more complex operations, Calcify provides a user-friendly interface that makes calculating quick and easy.


## Run Locally / Build

Clone the project

```bash
  git clone https://github.com/nishadimuhandiram/CALCIFY-ANDROID-CALCULATOR.git
```

Go to the project directory

```bash
  cd my-project
```


Build the APK (for debugging)

```bash
  ./gradlew assembleDebug
```
